allshop

allshop adalah prototipe user interface web e-commerce tipe business-to-consumer (B2C) yang dikembangkan oleh Ogi Yak pada 2016 - 2018 menggunakan Adobe XD Beta, HTML 5, CSS, JavaScript, serta Jquery berbasis user-centered-design (UCD) dan agile-software-development (ASD).
